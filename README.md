# Reyvora Analytics

Official company profile website developed using Laravel Framework.

## About

Reyvora Analytics is a company profile platform focused on data analytics, business intelligence, and digital transformation solutions.

## Features

- Home Page
- About Us
- Services
- Articles
- Contact Page
- Admin Dashboard
- Content Management

## Technologies

- Laravel
- PHP
- MySQL
- Bootstrap
- Blade Template Engine

## Installation

```bash
composer install
cp .env.example .env
php artisan key:generate
php artisan migrate --seed
php artisan serve
```
